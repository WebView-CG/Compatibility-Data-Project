CanIWebView App

CanIWebView distributes an open-source app, available on both the Google Play Store and Apple App Store, that allows you to test your content within the platform's webview. Within the app, you can customize WebView configurations to observe how your content behaves differently.

Since WebViews like Android WebView and WKWebView on iOS can behave very different, you should test in all environments.

This is especially useful for identifying whether an issue is reproducible, making it easier to share the necessary details when reporting bugs to the appropriate platform's issue tracker. If you discover a bug on either platform, you can submit it using the links below.

  • Google Chromium Issue Tracker
  • WebKit Issue Tracker (Bugzilla)

The source code and issue tracker for this application are available in the following GitHub repository:

  • Android App Source Code
  • iOS App Source Code

Do you use Microsoft Edge's WebView2?

We do not have this app distributed on any marketplace, but the source code is made available to download and compile locally for testing against on WebView2.

You can find the source code and issue tracker at our CanIWebView2 GitHub repository.

If you find a bug in the WebView you can submit an issue on the Microsoft Edge WebView2 Feedback repository on GitHub.
